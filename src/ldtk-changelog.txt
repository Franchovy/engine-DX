# Version 0.0.7

Dialog:
- Boolean "flip" (true = faces left, false = faces right)
- Integer (nullable) "saveNumber" - control order of bots that are rescued / REQUIRED for rescueable bots
- Boolean "save" - if on, bot is rescuable.
- Integer (nullable) "spriteNumber" - which sprite to use for this bot (only applicable to rescuable bots)
